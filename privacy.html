Política de Privacidad

Aplicación: bot ADS tu carro facil (herramienta interna de gestión de Meta Ads)  ·  Última actualización: 9 de julio de 2026

01 Qué es esta aplicación

Esta aplicación es una herramienta de uso interno que utiliza la API de Marketing de Meta para administrar, analizar y optimizar las cuentas publicitarias propias del operador de la aplicación. No está dirigida al público general ni se distribuye a terceros: la usan exclusivamente las personas autorizadas del negocio para gestionar sus propias campañas de Meta Ads.

02 Qué datos accedemos

A través de la API de Marketing de Meta, la aplicación accede únicamente a datos de administración publicitaria de las cuentas propias del operador:

  • Identificadores y configuración de cuentas publicitarias, campañas, conjuntos de anuncios y anuncios.
  • Métricas de rendimiento agregadas: gasto, impresiones, clics, conversaciones iniciadas y costo por resultado.
  • Identificadores de las Páginas de Meta conectadas y la configuración de enrutamiento de anuncios de clic a WhatsApp.

03 Qué NO recopilamos

La aplicación no accede, almacena ni procesa el contenido de las conversaciones de WhatsApp con los clientes finales, ni datos personales de las personas que responden a los anuncios (leads). Esas conversaciones ocurren directamente entre el asesor comercial y el cliente en WhatsApp, fuera del alcance de esta aplicación.

04 Para qué usamos los datos

Los datos de rendimiento publicitario se usan exclusivamente para fines internos de análisis y optimización: decidir si escalar, pausar o ajustar campañas propias, y generar reportes de desempeño para el equipo. No se usan para ningún otro propósito.

05 Almacenamiento y retención

Los datos de rendimiento permanecen en la plataforma de Meta y se consultan bajo demanda a través de la API. Cualquier extracción temporal para análisis interno se conserva únicamente el tiempo necesario para ese análisis y no se comparte fuera del equipo operador.

06 Con quién compartimos datos

No vendemos, alquilamos ni compartimos estos datos con terceros. El acceso está limitado a las personas del equipo autorizadas para administrar las cuentas publicitarias del negocio.

07 Seguridad

El acceso a la API se controla mediante tokens de autenticación de Meta, asignados a usuarios administrativos o de sistema dentro del Business Manager del operador, y revocables en cualquier momento desde la configuración del negocio en Meta.

08 Eliminación de datos

Para solicitudes de eliminación de datos, consulta nuestras Instrucciones de Eliminación de Datos o escríbenos directamente (ver contacto abajo).

09 Cambios a esta política

Esta política puede actualizarse ocasionalmente para reflejar cambios en cómo opera la aplicación. La fecha de "última actualización" en la parte superior indica la versión vigente.
